# AI Adventure

## Description
An interactive AI-driven text adventure game where you can input any action and the AI generates the story in real-time.

## Installation
Simply open `index.html` in a browser, or deploy to GitHub Pages.

## Usage
1. Enter your MiniMax API Key
2. Set the System Prompt (scene/character setting)
3. Start typing your actions and the AI will generate the story
